• text-indent
    • 必须应用于块级元素，不能应用在行内元素和replaced元素-例如image上
    • 该属性会被子类块级元素继承
    • 可以接收负值，首行文字突出
  • text-align
    • 必须应用于块级元素
    • does not control the alignment of elements, only their inline content.
    • text-align:justify —— the spacing between words and letters is adjusted so that each line is precisely the same length
  • line-height
    • refers to the distance between the baselines of lines of text rather than the size of the font
    • When applied to a block-level element, line-height defines the minimum distance between text baselines within that element.
    • 子元素继承自父元素的计算后的值
    • div {line-height: 1.5;} 使用1.5 * font-size 计算值作为最终的结果
  • vertical-align
    • applies only to inline elements and replaced elements such as images and form inputs
    • 百分比参考值为line-height
    • 如果给image, a form input, or another replaced element这些没有baseline的元素text-align:baseline，则该元素底部 aligned with the baseline of its parent
    • vertical-align: sub指的是该元素align相对于其父元素的baseline再低一点，低多少由浏览器决定，super则相反
    • text-bottom指的是行内文字的底部，对于replaced elements, or any other kinds of non-text elements,由父元素的font-size得出的一个默认的box，该元素底部和默认的box底部对齐
    • 百分比通过该元素的line-height进行计算，得出相对于baseline的偏移量，整数向上负数向下
    • It's important to realize that vertically aligned text does not become part of another line, nor does it overlap text in other lines
    • vertically aligned element can affect the height of the line
