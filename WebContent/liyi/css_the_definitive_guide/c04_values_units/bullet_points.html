• color
    1. rgb(100%,100%,100%) - rgb(0%,0%,0%)
    2. rgb(255,255,255) - rgb(0,0,0)
    3. #FFFFFF - #000000
    4. #RRGGBB简写为#RGB
    5. 超出范围的设定值被认为是边界值
    6. WEB-SAFE COLORS
      • rgb(40%,100%,80%) 能被20整除
      • rgb(0,204,153) 能被51整除的数
      • 00, 33, 66, 99, CC, and FF
  • Length Units
    • Absolute Length Units
      1. Inches (in)
      2. Centimeters (cm)
      3. Millimeters (mm)
      4. Points (pt)
      5. Picas (pc)
    • Relative Length Units
      1. EM, 当前元素font-size的倍数
      2. EX，当前元素字体中小写x的高度值，通常被浏览器解析为font-size的0.5em
      3. PIXEL
  • URL
    • body {background: url(http://www.pix.web/picture1.jpg);}
    • body {background: url(images/picture2.jpg);}
    • 注意url和()中间不能有空格
  • INHERIT
    Ordinarily, directly assigned styles override inherited styles, but inherit can reverse that behavior.
    即为inherit属性将使用继承下来的属性，而不是被指定的属性，当冲突的时候使用权重决定哪个样式被加载
