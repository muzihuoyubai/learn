• block-level Horizontal Formatting
    • margin--left/right, border--left/right, padding--left/right, width决定元素的水平的宽度
    • margin可以为负值，负值将缩短两个元素的距离
    • 七个值之和等于parent element或containing box的width
    • 当七个值被给定值后，加起来不等于containing box的width，margin-right强制被转换为auto属性，使得最后等式平衡
    • 当margin和width同时为auto属性时，margin被设置为0，width为可能的最大值
    • 百分比的设定值，百分比参考其父元素的width，borders cannot have percentage widths
    • 当replaced element width被设定为auto时，其宽度为替换后元素的真实宽度，例如img，会显示图片的实际宽度
  • block-level vertical Formatting
    • margin--top/bottom, border--top/bottom, padding--top/bottom, width决定元素的垂直高度
    • margin可以为负值，负值将缩短两个元素的距离
    • 七个值之和等于parent element或containing box的height
    • 百分比的设定值，百分比同样参考参考其父元素的width，borders cannot have percentage widths
