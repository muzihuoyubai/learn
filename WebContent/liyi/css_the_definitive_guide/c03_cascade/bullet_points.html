• specificity权重，当有多个选择器同时选中某元素并且操作的样式有冲突的时候，通过权重进行判断哪个样式最终被加载
    1. !important，带有important的归为一组进行权重比较与没有important的样式冲突的时候，总是会加载important的样式，不考虑权重
    2. 行内样式，即html标签中的style=""，1000
    3. ID选择器，0100
    4. 类、属性、伪类选择器，0010
    5. 元素、伪元素选择器，0001
    6. 关系选择器以及全局选择器无权重
    • 示例
    • html > body table tr[id="totals"] td ul > li {color: maroon;} /* 0,0,1,7 */
    • hli#answer {color: navy;} /* 0,1,0,1 (winner) */
    • #meadow{color:red} 权重为0100，*[id='meadow']{color:red}权重为0010
